
# 语音合成功能需求文档

## 1. 功能概述

语音合成功能（Text-to-Speech，TTS）允许用户输入文本内容，系统将其转换为AI语音输出，支持多种语言和声音风格选择。

## 2. 功能需求

### 2.1 核心功能

| 需求编号 | 需求描述 | 优先级 |
| :--- | :--- | :--- |
| REQ-TTS-001 | 用户可输入待合成的文本内容 | 高 |
| REQ-TTS-002 | 用户可选择目标语言（与文案生成共用语言列表） | 高 |
| REQ-TTS-003 | 用户可选择语音风格/音色 | 高 |
| REQ-TTS-004 | 用户可调整语音语速（快/中/慢） | 中 |
| REQ-TTS-005 | 用户可调整语音音量 | 中 |
| REQ-TTS-006 | 支持在线播放合成的语音 | 高 |
| REQ-TTS-007 | 支持下载合成的语音文件（MP3格式） | 中 |
| REQ-TTS-008 | 显示语音合成进度状态 | 高 |
| REQ-TTS-009 | 支持语音合成历史记录查询 | 低 |

## 4. API接口设计

### 4.1 合成语音接口

| 属性 | 值 |
| :--- | :--- |
| **接口地址** | `/api/ai/tts` |
| **请求方法** | `POST` |
| **Content-Type** | `application/x-www-form-urlencoded` |

**请求参数**:

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| userId | Number | 是 | - | 用户ID |
| text | String | 是 | - | 待合成文本（最大5000字符） |
| language | String | 是 | 简体中文 | 目标语言 |
| voice | String | 否 | female | 语音风格 |
| speed | Number | 否 | 1.0 | 语速（0.5-2.0） |
| volume | Number | 否 | 1.0 | 音量（0.1-1.0） |

**成功响应**:

```json
{
  "success": true,
  "data": {
    "audioUrl": "https://example.com/audio/xxx.mp3",
    "duration": 15.5,
    "textLength": 120
  }
}
```

| 字段 | 类型 | 说明 |
| :--- | :--- | :--- |
| success | Boolean | 是否成功 |
| data.audioUrl | String | 合成音频文件URL |
| data.duration | Number | 音频时长（秒） |
| data.textLength | Number | 合成文本长度 |

**失败响应**:

```json
{
  "success": false,
  "message": "文本内容不能为空"
}
```

### 4.2 获取语音风格列表接口

| 属性 | 值 |
| :--- | :--- |
| **接口地址** | `/api/ai/tts/voices` |

| :--- | :--- | :--- |
| tts | 语音合成 | Text-to-Speech |
| tts.textLabel | 文本内容 | Text Content |
| tts.textPlaceholder | 请输入需要合成语音的文本... | Enter text to synthesize... |
| tts.languageLabel | 目标语言 | Target Language |
| tts.voiceLabel | 语音风格 | Voice Style |
| tts.speedLabel | 语速 | Speed |
| tts.volumeLabel | 音量 | Volume |
| tts.slow | 慢 | Slow |
| tts.normal | 中 | Normal |
| tts.fast | 快 | Fast |
| tts.play | 播放 | Play |

## 8. 注意事项

1. **文本长度限制**: 建议限制单次合成文本最大长度为5000字符
2. **音频格式**: 返回MP3格式，确保多端兼容性
3. **错误处理**: 需要处理网络异常、文本为空、音频生成失败等场景
4. **播放状态管理**: 需要维护播放/暂停/停止状态，避免重复播放
5. **权限处理**: 语音合成功能可能需要消耗积分，需与现有积分系统对接
