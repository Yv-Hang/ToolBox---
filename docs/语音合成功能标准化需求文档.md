## 1. 功能概述

### 1.1 功能说明

语音合成功能（Text-to-Speech，TTS）基于小米 MiMo V2.5 TTS 技术，允许用户输入文本内容，系统将其转换为自然流畅的AI语音输出。支持三种模式：
- **预置音色模式**: 使用内置精品音色
- **音色设计模式**: 通过文本描述定制音色
- **音色复刻模式**: 基于音频样本复刻任意音色

### 1.2 支持的模型

| 模型名称 | Model ID | 功能说明 |
|---|---|---|
| 预置音色合成 | `mimo-v2.5-tts` | 使用预置精品音色，支持唱歌模式 |
| 音色设计合成 | `mimo-v2.5-tts-voicedesign` | 通过文本描述自动生成音色 |
| 音色复刻合成 | `mimo-v2.5-tts-voiceclone` | 基于音频样本精准复刻音色 |

### 3.5 交互逻辑规范

#### 3.5.1 表单验证

| 验证项 | 触发时机 | 验证规则 | 错误提示 |
|---|---|---|---|
| 文本内容 | 失去焦点 / 提交 | 非空，≤5000字符 | 请输入合成文本（最多5000字符） |
| 音色描述 | 失去焦点 / 提交 | 非空，≤200字符 | 请输入音色描述（最多200字符） |
| 音频样本 | 文件选择时 | MP3/WAV格式，≤10MB | 请上传MP3或WAV格式文件，不超过10MB |

#### 3.5.2 合成流程

1. 用户填写表单 → 2. 点击"开始合成" → 3. 显示加载状态 → 4. 合成成功显示播放面板 → 5. 支持播放/下载

#### 3.5.3 播放状态管理

| 状态 | 说明 |
|---|---|
| idle | 空闲状态，等待合成 |
| synthesizing | 正在合成中 |
| ready | 合成完成，等待播放 |
| playing | 正在播放 |
| paused | 暂停播放 |
| stopped | 停止播放 |

## 4. 后端接口规范

### 4.1 通用响应格式

所有接口统一使用以下响应格式：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 4.2 获取预置音色列表

| 属性 | 值 |
|---|---|
| **接口路径** | `/api/tts/voices` |
| **请求方法** | `GET` |
| **Content-Type** | `application/json` |

**请求参数**: 无

**成功响应 (200)**:

```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "voiceId": "冰糖",
      "name": "冰糖",
      "language": "中文",
      "gender": "female",
      "description": "标准中文女声"
    },
    {
      "voiceId": "Mia",
      "name": "Mia",
      "language": "英文",
      "gender": "female",
      "description": "标准英文女声"
    }
  ]
}
```

**响应字段说明**:

| 字段 | 类型 | 说明 |
|---|---|---|
| voiceId | String | 音色ID（用于接口调用） |
| name | String | 音色显示名称 |
| language | String | 适用语言 |
| gender | String | 性别: male/female |
| description | String | 音色描述 |

### 4.3 预置音色合成接口

| 属性 | 值 |
|---|---|
| **接口路径** | `/api/tts/synthesize` |
| **请求方法** | `POST` |
| **Content-Type** | `application/json` |

**请求参数**:

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|
| model | String | 是 | - | 固定值: "mimo-v2.5-tts" |
| text | String | 是 | - | 待合成文本（最大5000字符） |
| voice | String | 否 | "冰糖" | 预置音色ID |
| styleInstruction | String | 否 | "" | 自然语言风格描述 |
| audioFormat | String | 否 | "wav" | 音频格式: wav/mp3/pcm16 |
| userId | String | 否 | "" | 用户ID（用于记录） |

**请求示例**:

```json
{
  "model": "mimo-v2.5-tts",
  "text": "(磁性)夜已经深了，城市还在呼吸。",
  "voice": "冰糖",
  "styleInstruction": "温柔的语气，语速稍慢",
  "audioFormat": "wav"
}
```

**成功响应 (200)**:

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "taskId": "tts_20260519_abc123",
    "audioUrl": "https://example.com/audio/tts_20260519_abc123.wav",
    "audioData": "base64encodedstring...",
    "duration": 15.3,
    "textLength": 25,
    "format": "wav",
    "sampleRate": 24000
  }
}
```

**响应字段说明**:

| 字段 | 类型 | 说明 |
|---|---|---|
| taskId | String | 任务ID |
| audioUrl | String | 音频文件URL |
| audioData | String | Base64编码的音频数据 |
| duration | Number | 音频时长（秒） |
| textLength | Number | 合成文本长度 |
| format | String | 音频格式 |
| sampleRate | Number | 采样率 |

## 6. 数据验证规则

### 6.1 前端验证规则

| 字段 | 验证规则 | 错误码 |
|---|---|---|
| text | 非空，长度 1-5000 | VALIDATION_TEXT_EMPTY, VALIDATION_TEXT_TOO_LONG |
| voiceDescription | 非空，长度 1-200 | VALIDATION_VOICE_DESC_EMPTY, VALIDATION_VOICE_DESC_TOO_LONG |
| audioSample | 文件类型: mp3/wav，大小 ≤10MB | VALIDATION_FILE_TYPE, VALIDATION_FILE_SIZE |

### 6.2 后端验证规则

| 参数 | 验证规则 | 错误码 |
|---|---|---|
| model | 必须为指定模型ID之一 | ERR_INVALID_MODEL |
| text | 非空，长度 ≤5000 | ERR_TEXT_EMPTY, ERR_TEXT_TOO_LONG |
| voice | 必须在预置音色列表中 | ERR_INVALID_VOICE |
| voiceDescription | 非空，长度 ≤200 | ERR_VOICE_DESC_EMPTY |
| audioFormat | 必须为 wav/mp3/pcm16 之一 | ERR_INVALID_FORMAT |
| audioSample | 必须为 mp3/wav，大小 ≤10MB | ERR_INVALID_AUDIO_SAMPLE |
